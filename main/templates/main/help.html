Commands


Forward

The forward command will be used to prompt Qbot to move forward a "certain distance" based on the number the user has prompted.

example - "Hey Qbot forward five"
- Qbot will recieve forward as the command word and five as the distance to move forward 5 "meters"


Backward

The backward command will be used to prompt Qbot to move backward a "certain distance" based on the number the user has prompted.

example - "Hey Qbot backward 3"
- Qbot will recieve backward as the command word and three as the distance to move backward 3 "meters"


Right

The right command will be used to prompt the Qbot to turn right 90 degrees, this is a manual command to help redirect the Qbot away from walls or other objects and back on its original path.

example - "Hey Qbot right"
- Qbot will recieve right as the command word and will complete a 90 degree turn to the right


Left

The left command will be used to prompt the Qbot to turn left 90 degrees, this is a manual command to help redirect the Qbot away from walls or other objects and back on its original path.

example - "Hey Qbot left"
- Qbot will recieve left as the command word and will complete a 90 degree turn to the left


Follow

The follow command will be used to prompt the Qbot to follow A colour designated by the user, it requires the command prompt and a colour prompt (Red, Green, Black) to recognise its command.

example - "Hey Qbot follow red"
- Qbot will recieve follow as the command prompt and red as the directive, the QBot will follow a red line designed on the floor
